# Beach Wedding Invite

A premium, animated wedding invitation website built with React, GSAP, Lenis smooth scroll, and Tailwind CSS.

## Features

- Cinematic hero with parallax background
- Smooth scroll powered by Lenis
- Horizontal scroll event cards (Mehendi, Haldi, Cocktail, Shaadi, Reception)
- Meet the Bride & Groom section with slide-in animations
- Photo gallery with hover effects
- Couple's message with decorative typography
- WhatsApp RSVP integration
- Live countdown timer
- Things to Know info cards
- Fully responsive, mobile-first design

## Tech Stack

- **React 19** + **Vite**
- **GSAP** + **ScrollTrigger** — scroll-driven animations
- **Lenis** — smooth scroll
- **Tailwind CSS v4** — styling
- **Google Fonts** — Cormorant Garamond + Montserrat

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
